1. Add dark_web config to `backend/config/services.php`:
   ```php
   'dark_web' => [
       'api_key' => env('DARK_WEB_API_KEY'),
       'base_url' => env('DARK_WEB_API_URL', 'https://leakcheck.io/api/v2'),
   ],
   ```

2. Create `backend/app/Services/DarkWebProviderService.php`:
   - `search(string $query, string $type): array` method
   - Uses `Http::withHeaders(['X-API-Key' => config('services.dark_web.api_key')])->timeout(10)->retry(2, 500, fn($e) => $e instanceof ConnectionException)`
   - GET request to `config('services.dark_web.base_url') . '/query/' . urlencode($query)` with `['type' => $type]`
   - Calls `$response->throw()` on non-2xx
   - Returns `['found' => $body['found'] ?? 0, 'results' => $this->normalizeResults($body['result'] ?? [])]`
   - `normalizeResults()` maps each item to: email, password (masked), username, source (from source.name), breach_date (from source.breach_date), first_name, last_name, phone, fields
   - `maskPassword(?string $password): ?string` -- null returns null, empty returns null, otherwise first min(3, strlen) chars + asterisks for the rest

3. Create `backend/app/Http/Requests/DarkWebSearchRequest.php`:
   - Rules: `'query' => 'required|string|max:320'`, `'type' => 'required|in:email,domain'`
   - `withValidator()` after-hook: if type=email validate with filter_var FILTER_VALIDATE_EMAIL, if type=domain validate with regex `/^[a-zA-Z0-9]([a-zA-Z0-9-]*[a-zA-Z0-9])?(\.[a-zA-Z]{2,})+$/`

4. Create `backend/app/Http/Controllers/DarkWeb/SearchController.php`:
   - Invokable controller, accepts DarkWebSearchRequest
   - Gets credit from `$request->attributes->get('credit')`
   - Wraps `app(DarkWebProviderService::class)->search()` in try/catch
   - On catch: refund credit with `DB::table('credits')->where('id', $credit->id)->increment('remaining')`, then `$credit->refresh()`, return 502 with message "Something went wrong. No credit was deducted." and credits object
   - On success: create SearchLog with module='dark_web', return 200 with data + credits object
   - Credits object: `{ remaining: $credit->remaining, limit: $credit->limit, resets_at: now('UTC')->addDay()->startOfDay()->toIso8601String() }`

5. Add route to `backend/routes/api.php`:
   - Import `use App\Http\Controllers\DarkWeb\SearchController as DarkWebSearchController;`
   - Inside the existing `Route::middleware('auth:sanctum')` group, add: `Route::post('/dark-web/search', DarkWebSearchController::class)->middleware('deduct-credit');`
