<task type="auto">
  <name>Task 2: Configure Sanctum SPA middleware, CORS, User model, migrations, and run migrate</name>
  <files>backend/bootstrap/app.php, backend/config/cors.php, backend/app/Models/User.php, backend/database/migrations/xxxx_add_oauth_columns_to_users_table.php</files>
  <action>
1. Edit backend/bootstrap/app.php -- add statefulApi() middleware:
   ```php
   ->withMiddleware(function (Middleware $middleware) {
       $middleware->statefulApi();
   })
   ```
   This enables Sanctum's SPA cookie-based auth (session + CSRF middleware on API routes for stateful requests).

2. Edit backend/config/cors.php:
   ```php
   return [
       'paths' => ['api/*', 'sanctum/csrf-cookie', 'login', 'logout', 'register'],
       'allowed_methods' => ['*'],
       'allowed_origins' => [env('FRONTEND_URL', 'http://localhost:5173')],
       'allowed_origins_patterns' => [],
       'allowed_headers' => ['*'],
       'exposed_headers' => [],
       'max_age' => 0,
       'supports_credentials' => true,
   ];
   ```
   CRITICAL: Do NOT use wildcard '*' for allowed_origins -- it breaks credentialed requests.

3. Edit backend/app/Models/User.php:
   - Add `use Laravel\Sanctum\HasApiTokens;` import
   - Add `HasApiTokens` to the use traits line
   - Add `'oauth_provider', 'oauth_id', 'avatar_url'` to $fillable
   - Add `'oauth_provider', 'oauth_id'` to $hidden
   - Ensure casts includes `'email_verified_at' => 'datetime'` and `'password' => 'hashed'`

4. Create migration for OAuth columns:
   ```bash
   php artisan make:migration add_oauth_columns_to_users_table --table=users
   ```
   In the up() method:
   ```php
   $table->string('oauth_provider', 50)->nullable()->after('remember_token');
   $table->string('oauth_id', 255)->nullable()->after('oauth_provider');
   $table->string('avatar_url', 500)->nullable()->after('oauth_id');
   $table->index(['oauth_provider', 'oauth_id']);
   ```
   In the down() method:
   ```php
   $table->dropIndex(['oauth_provider', 'oauth_id']);
   $table->dropColumn(['oauth_provider', 'oauth_id', 'avatar_url']);
   ```

5. Run all migrations:
   ```bash
   php artisan migrate
   ```
   This creates: users, password_reset_tokens, sessions, cache, jobs, personal_access_tokens tables, plus adds OAuth columns.

6. Verify the database schema:
   ```bash
   php artisan db:show --counts
   ```
