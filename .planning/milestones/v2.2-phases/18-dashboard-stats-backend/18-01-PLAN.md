**Stale-cache pattern (use for ALL three methods):**
Do NOT use `Cache::remember()` -- it deletes expired entries so there is nothing to fall back to. Instead use manual `Cache::get()` + `Cache::put()`:
```php
$cacheKey = 'dashboard_counts';
$cached = Cache::get($cacheKey);
try {
    $fresh = $this->fetchCounts();
    Cache::put($cacheKey, $fresh, now()->addMinutes(5));
    return $fresh;
} catch (OpenCtiConnectionException $e) {
    if ($cached !== null) {
        return $cached;
    }
    throw $e;
}
```

**Method 1: `getCounts(): array`**
- Cache key: `dashboard_counts`
- Calls private `fetchCounts()` which makes 4 sequential GraphQL queries (one per entity type)
- Entity types: `['IPv4-Addr', 'Domain-Name', 'Url', 'Email-Addr']`
- Human-readable labels: `['IP Addresses', 'Domains', 'URLs', 'Email Addresses']`
- Each query uses `pageInfo.globalCount` with `first: 1` to avoid fetching actual node data:
```graphql
query ($filters: FilterGroup) {
    stixCyberObservables(filters: $filters, first: 1) {
        pageInfo {
            globalCount
        }
    }
}
```
- Filter variable per type:
```php
$variables = [
    'filters' => [
        'mode' => 'and',
        'filters' => [
            [
                'key' => 'entity_type',
                'values' => [$entityType],
                'operator' => 'eq',
                'mode' => 'or',
            ],
        ],
        'filterGroups' => [],
    ],
];
```
- Returns array of 4 objects: `[['entity_type' => 'IPv4-Addr', 'label' => 'IP Addresses', 'count' => 1234], ...]`
- Extract count via: `$data['stixCyberObservables']['pageInfo']['globalCount'] ?? 0`

**Method 2: `getIndicators(): array`**
- Cache key: `dashboard_indicators`
- Calls private `fetchIndicators()` which queries 10 most recent observables:
```graphql
{
    stixCyberObservables(first: 10, orderBy: created_at, orderMode: desc) {
        edges {
            node {
                id
                entity_type
                observable_value
                x_opencti_score
                created_at
            }
        }
    }
}
```
- No variables needed (no filters).
- Normalize response: map edges to flat array of `['id' => ..., 'value' => $node['observable_value'], 'entity_type' => ..., 'score' => $node['x_opencti_score'] ?? null, 'created_at' => ...]`

**Method 3: `getCategories(): array`**
- Cache key: `dashboard_categories`
- Calls private `fetchCategories()` which queries 500 recent observables with objectLabel only:
```graphql
{
    stixCyberObservables(first: 500, orderBy: created_at, orderMode: desc) {
        edges {
            node {
                objectLabel {
                    value
                }
            }
        }
    }
}
```
- Aggregate labels in PHP:
```php
$labelCounts = [];
foreach ($edges as $edge) {
    foreach ($edge['node']['objectLabel'] ?? [] as $label) {
        $name = $label['value'] ?? '';
        if ($name !== '') {
            $labelCounts[$name] = ($labelCounts[$name] ?? 0) + 1;
        }
    }
}
arsort($labelCounts);
$top6 = array_slice($labelCounts, 0, 6, true);
return array_map(
    fn (string $name, int $count) => ['label' => $name, 'count' => $count],
    array_keys($top6),
    array_values($top6),
);
```

<?php

namespace App\Http\Controllers\Dashboard;

use App\Exceptions\OpenCtiConnectionException;
use App\Http\Controllers\Controller;
use App\Services\DashboardService;
use Illuminate\Http\JsonResponse;

class CountsController extends Controller
{
    public function __invoke(): JsonResponse
    {
        try {
            $counts = app(DashboardService::class)->getCounts();

            return response()->json([

        }

        return response()->json(['data' => $counts]);
    }
}
