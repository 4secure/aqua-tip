**Inline sub-components:**

1. **StatCard({ label, count, color, loading, error })** -- glass-card with label, count (toLocaleString), and "Live from OpenCTI" indicator with green live-dot. Loading state: animate-pulse placeholder. Error state: "Failed to load" in text-red.

2. **AttackChart({ categories, activeFilter, onFilterChange })** -- Horizontal bar chart using `useChartJs`. Config uses `categories.map(c => c.label)` for labels, `categories.map(c => c.count)` for data. Bar colors from CATEGORY_COLORS array by index. Active filter bar gets full opacity, others get 40% opacity (use `backgroundColor` array where active index gets `CATEGORY_COLORS[i]` and others get `CATEGORY_COLORS[i] + '40'`). Chart.js `onClick` handler: get clicked element index, derive label from `categories[index].label`, call `onFilterChange(clickedLabel === activeFilter ? null : clickedLabel)`. Chart options: `indexAxis: 'y'`, `responsive: true`, `maintainAspectRatio: false`, legend hidden, tooltip styled with dark theme (`backgroundColor: '#161822'`, `borderColor: '#2A2D3E'`, `borderWidth: 1`).

3. **IndicatorsTable({ indicators, activeFilter, onClearFilter })** -- Table with columns: Type (badge), Value, Labels (chips), Date. Above the table, if `activeFilter` is truthy, show a filter chip: `<span className="chip chip-violet text-xs">{activeFilter} <button onClick={onClearFilter} className="ml-1">x</button></span>`. Filter the indicators: `const displayed = activeFilter ? indicators.filter(ind => (ind.labels || []).includes(activeFilter)) : indicators`. Slice to 8 rows: `.slice(0, 8)`. Type badge uses TYPE_BADGE_COLORS for coloring. Labels rendered as small chips. Date formatted as relative or short date.

4. **CreditWidget({ credits, loading, error })** -- glass-card with title "Credit Balance", icon (coins). Shows `credits.remaining / credits.limit` with a progress bar (width = `(remaining/limit)*100%`). Progress bar color: green if ratio >= 0.5, amber if ratio >= 0.2, red otherwise. Show `credits.resets_at` formatted as "Resets: {date}". Include the existing CreditBadge component for the ratio chip. Loading/error states like StatCard.

5. **RecentSearchesWidget({ searches, loading, error })** -- glass-card with title "Recent Searches", icon (clock). Show up to 5 most recent items from `searches.slice(0, 5)`. Each row: query text (font-mono, truncate), type badge using TYPE_BADGE_COLORS, relative timestamp. Empty state: "No searches yet" text. Loading/error states.

6. **GuestCta({ icon, message })** -- glass-card centered with icon in violet/10 bg, message text, and "Sign In" Link to `/login` with `btn-primary` class. Match the pattern from ThreatSearchPage guest CTA styling.
