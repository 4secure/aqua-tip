## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing (Tailwind `4` = 16px used in map widget positioning) |
| lg | 24px | Section padding (AppLayout content padding: 24px via `p-6`) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Map viewport uses `-m-6` (negative 24px margin) to negate AppLayout's `p-6` padding, achieving edge-to-edge layout. Map widget overlays use `top-4 left-4` (16px) and `bottom-4 right-4` (16px) for inset positioning.

## Color

No new colors introduced. Existing map color contract preserved:

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0A0B10` (primary) | Page background (hidden behind map) |
| Secondary (30%) | `#0F1117` (surface) at 60% opacity | Map overlay widget backgrounds (glassmorphism) |
| Accent (10%) | `#00E5FF` (cyan) | Connection status indicator, pulse marker default color, live feed highlight |
| Destructive | `#FF3B5C` (red) | Critical severity pulse markers, malware indicators |

Additional semantic colors active on map widgets:
- `#00C48C` (green): Safe/connected status dot, low-severity indicators
- `#FFB020` (amber): Warning-level indicators, medium severity
- `#7A44E4` (violet): Not used on map widgets in this phase

Accent reserved for: SSE connection status badge, default pulse marker color, feed entry hover highlight, counter value emphasis.

Map tile background: `#262626` (hardcoded in ThreatMapPage inline style, not a design token).

## Copywriting Contract

This phase introduces no new UI copy. It is a route swap that preserves existing content. Existing map widget copy contracts preserved:

| Element | Copy |
|---------|------|
| Connection status (connected) | "LIVE" (green dot + label in ThreatMapStatus) |
| Connection status (disconnected) | "OFFLINE" (red dot + label in ThreatMapStatus) |
| Empty state (no events) | Feed widget shows empty list; counters show 0 — no explicit empty state message |
| Error state | Connection status changes to "OFFLINE" — no error modal or toast |
| Primary CTA | None — this phase is a passive display page with no user-initiated actions beyond feed click-to-pan |

No destructive actions exist in this phase.

## Interaction Contract

All interactions are preserved from the existing ThreatMapPage — no modifications:

| Interaction | Behavior | Source |
|-------------|----------|--------|
| Feed entry click | Map flies to event coordinates at zoom 6, highlight pulse appears | ThreatMapPage handleEventClick (line 65) |

| Route redirect | `/threat-map` navigates to `/dashboard` with `replace` (no back-button loop) | D-02 |
