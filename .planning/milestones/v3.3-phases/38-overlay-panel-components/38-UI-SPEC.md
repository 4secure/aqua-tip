## Interaction Contract

### Collapse / Expand Animation

| Property | Value |
|----------|-------|
| Library | Framer Motion `AnimatePresence` + `motion.div` |
| Left panel enter | `initial={{ x: -20, opacity: 0 }}` -> `animate={{ x: 0, opacity: 1 }}` |
| Left panel exit | `exit={{ x: -20, opacity: 0 }}` |
| Right panel enter | `initial={{ x: 20, opacity: 0 }}` -> `animate={{ x: 0, opacity: 1 }}` |
| Right panel exit | `exit={{ x: 20, opacity: 0 }}` |
| Transition | `type: 'spring', stiffness: 300, damping: 30` |
| Toggle button | Always visible regardless of collapse state |

### Event Propagation Isolation

Every panel container (`LeftOverlayPanel`, `RightOverlayPanel`, `PanelToggle`) must apply these handlers on the outermost `div`:

```
onPointerDown={e => e.stopPropagation()}
onWheel={e => e.stopPropagation()}
onClick={e => e.stopPropagation()}
onDoubleClick={e => e.stopPropagation()}
onTouchStart={e => e.stopPropagation()}
```

This prevents Leaflet from receiving click (pan), double-click (zoom), wheel (zoom), and touch (mobile pan/zoom) events when interacting with panels.

### State Management

| State | Location | Type | Default |
|-------|----------|------|---------|
| `panelsCollapsed` | ThreatMapPage | `useState(false)` | `false` (panels visible on load) |
| `counts` + `countsLoading` + `countsError` | LeftOverlayPanel | `useState` + `useEffect` fetch | Fetch from `/api/dashboard/counts` on mount |
| `indicators` + `indicatorsLoading` + `indicatorsError` | RightOverlayPanel | `useState` + `useEffect` fetch | Fetch from `/api/dashboard/indicators` on mount |

Note: `panelsCollapsed` is NOT persisted to localStorage in this phase. That is Phase 39 (TOGGLE-04).

## Loading & Error States

### Stat Cards

| State | Visual |
|-------|--------|
| Loading | Each stat row shows label normally, count replaced by `h-4 w-12 bg-surface-2 rounded animate-pulse` skeleton |
| Error | Each stat row shows label normally, count shows "---" in `text-text-muted` |
| Success | Count displayed as locale-formatted number in `font-mono font-semibold text-text-primary` |

### Indicators Table

| State | Visual |
|-------|--------|
| Loading | 5 skeleton rows with alternating widths (60%, 80%, 50%, 70%, 65%) using `bg-surface-2 rounded animate-pulse` |
| Error | Centered text "Failed to load indicators" in `text-xs text-text-muted`, no retry button |
| Empty | Centered text "No indicators found" in `text-xs text-text-muted` |
| Success | Scrollable list of indicator rows with type badge, value, and relative time |
