  - No second filter for entity_type (removed)

**NEW method detectHashFilterKey($query):**
```php
private function detectHashFilterKey(string $query): ?string
{
    $clean = strtolower(trim($query));
    if (!ctype_xdigit($clean)) {
        return null;
    }
    return match (strlen($clean)) {
        32 => 'hashes.MD5',
        40 => 'hashes.SHA-1',
        64 => 'hashes.SHA-256',
        default => null,
    };
}
```

- `'query' => $query` (was `'ip' => $ip`)
- Add `'detected_type' => $observable['entity_type'] ?? null,` as second field
- All other fields identical
