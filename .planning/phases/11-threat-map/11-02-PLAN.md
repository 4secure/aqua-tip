  <action>
    Create `frontend/src/hooks/useThreatStream.js`:

    Exports `useThreatStream()` returning `{ events, counters, countryCounts, typeCounts, connected }`.

    **Snapshot loading (on mount):**
    - Fetch `GET /api/threat-map/snapshot` via axios (already configured with interceptors)
    - Initialize `events` state from `response.data.data.events` (array, max 50 items)
    - Initialize counters from `response.data.data.counters`
    - Derive `countryCounts` (object: `{ countryCode: { name, count } }`) and `typeCounts` (object: `{ type: count }`) from events

    **SSE connection (after snapshot loads):**
    - Create `new EventSource('/api/threat-map/stream', { withCredentials: true })`
    - `es.onopen`: set `connected = true`
    - `es.onmessage`: parse JSON, prepend to events (immutable: `[newEvent, ...prev].slice(0, 50)`), update counters/countryCounts/typeCounts immutably
    - `es.onerror`: set `connected = false` (EventSource auto-reconnects; the server sends `retry: 10000` for 10s interval)
    - Cleanup: `es.close()` on unmount

    **Counter aggregation logic:**
    - `threats`: total event count
    - `countries`: count of unique countryCode values in events
    - `types`: count of unique type values in events
    - `countryCounts`: aggregate by countryCode, sorted by count descending, top 5
    - `typeCounts`: aggregate by type, for donut chart

    Use `useCallback` for the aggregation function. Use `useRef` for EventSource instance. All state updates must be immutable (spread, not push).

    **CSS animations** — append to `frontend/src/styles/animations.css`:

    ```css
    @keyframes mapEventPulse {
      0% { transform: scale(0.3); opacity: 1; }
      100% { transform: scale(3); opacity: 0; }
    }

    .map-event-pulse {
      border-radius: 50%;
      animation: mapEventPulse 1.5s ease-out forwards;
      pointer-events: none;
    }

    .map-event-pulse--red { background: rgba(255, 59, 92, 0.6); box-shadow: 0 0 12px rgba(255, 59, 92, 0.4); }
    .map-event-pulse--amber { background: rgba(255, 176, 32, 0.6); box-shadow: 0 0 12px rgba(255, 176, 32, 0.4); }
    .map-event-pulse--violet { background: rgba(122, 68, 228, 0.6); box-shadow: 0 0 12px rgba(122, 68, 228, 0.4); }
    .map-event-pulse--cyan { background: rgba(0, 229, 255, 0.6); box-shadow: 0 0 12px rgba(0, 229, 255, 0.4); }

    .map-event-pulse--highlight {
      background: rgba(255, 255, 255, 0.8);
      box-shadow: 0 0 20px rgba(255, 255, 255, 0.6);
      animation: mapEventPulse 2s ease-out forwards;
    }
    ```
