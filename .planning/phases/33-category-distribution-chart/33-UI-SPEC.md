## Component Inventory

### New Component: CategoryDistributionChart

| Property | Specification |
|----------|---------------|
| Container | `div` with classes `bg-surface/60 border border-border backdrop-blur-sm rounded-xl p-6` |
| Height | 180px fixed (`h-[180px]`) |
| Chart type | Stacked area (Chart.js `line` type with `fill: true` and stacking) |
| X-axis | Hours 0h through 23h (24 ticks). Label format: `"0h"`, `"1h"`, ... `"23h"` |
| Y-axis | Report count (integer). Begins at 0. Grid lines at `rgba(30, 32, 48, 0.125)` |
| Datasets | One dataset per category. Each uses `fill: 'origin'` with `stack: 'stack0'` for stacking. `tension: 0.4` for smooth curves. `pointRadius: 0` (no dots). `borderWidth: 2`. |
| Legend | Custom HTML legend below chart canvas. Horizontal flex-wrap row of category swatches (8px circles) + 12px labels. Max 8 items visible. |
| Click behavior | `onClick` handler on chart resolves clicked dataset label, calls `updateParam('label', categoryId)` using the same mechanism as category chip clicks. Clicking the active category clears the filter. |
| Tooltip | Single-point tooltip showing category name (title) and count for that hour (body). Format: `"{count} reports"`. Background/border/text colors per Chart Chrome table above. Corner radius 8px, padding 8px. |
| Animation | Chart.js default transitions (`animation.duration: 750`). No custom animation code. |

### Placement in ThreatNewsPage

```
[Toolbar: date selector + category dropdown + search]
   --- 16px gap (gap-4) ---
[CategoryDistributionChart card]  <-- NEW
   --- 16px gap (gap-4) ---
[Report list / empty state]
```

### Visibility Rules

| Condition | Chart Behavior |
|-----------|---------------|
| Reports exist with at least 1 categorized item | Chart visible |
| Reports exist but none have categories | Chart hidden entirely (no card rendered) |
| No reports for selected date | Chart hidden entirely (existing empty state handles messaging) |
| Date changes | Chart re-computes from new `items`, Chart.js animates transition |
| Category filter active via `?label=` | Chart still visible, filtered category area highlighted (full opacity), others dimmed (15% opacity fill) |

## Interaction States

| State | Visual Treatment |
|-------|-----------------|
| Default (no filter) | All category areas at 35% fill opacity |
| Category filter active | Active category area at 60% fill opacity, all others at 15% fill opacity |
| Hover on chart area | Chart.js default hover — tooltip appears, hovered point area slightly brightened |
| Date loading | Chart area shows existing data until new data arrives, then Chart.js animates the transition (no skeleton, no flicker) |
| Legend item alignment | Horizontal flex-wrap, gap 16px between items, gap 4px between swatch and label |
