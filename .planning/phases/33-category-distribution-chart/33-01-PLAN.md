**IMPORTANT for color consistency:** The chart's hash function must map category names to the SAME visual color as the ThreatNewsPage chips. Replicate the EXACT same hash algorithm from ThreatNewsPage:
```javascript
function chartColorForCategory(labelValue) {
  let hash = 0;
  for (let i = 0; i < labelValue.length; i++) {
    hash = ((hash << 5) - hash) + labelValue.charCodeAt(i);
    hash |= 0;
  }
  return CHART_HEX_COLORS[Math.abs(hash) % CHART_HEX_COLORS.length];
}
```

Note: Since CHART_HEX_COLORS has 6 entries vs CATEGORY_COLORS' 5, the modulo will produce different slot indices for some categories. This is acceptable because the chart uses its own richer palette. The visual tone families still align (both use violet, cyan, amber, red).

**2. Hex-to-rgba helper:**
```javascript
function hexToRgba(hex, alpha) {
  const r = parseInt(hex.slice(1, 3), 16);
  const g = parseInt(hex.slice(3, 5), 16);
  const b = parseInt(hex.slice(5, 7), 16);
  return `rgba(${r}, ${g}, ${b}, ${alpha})`;
}
```
