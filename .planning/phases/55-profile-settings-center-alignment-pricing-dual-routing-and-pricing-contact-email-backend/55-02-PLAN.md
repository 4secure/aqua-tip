
4. Replace the entire `handleSubmit` function body with:
```jsx
const handleSubmit = async (e) => {
  e.preventDefault();
  setSending(true);
  setError(null);
  try {
    await csrfCookie();
    await apiClient.post('/api/enterprise/contact', {
      ...form,
      plan_name: planName,
    });
    setSent(true);
  } catch (err) {
    setError(err.message || 'Failed to send message. Please try again.');
  } finally {
    setSending(false);
  }
};
```

5. Add an error message display in the form, right before the `<div className="flex gap-3 pt-1">` buttons div:
```jsx
{error && (
  <p className="text-red text-sm font-mono">{error}</p>
)}
```

IMPORTANT: The `csrfCookie()` call before the POST is critical — unauthenticated users may not have the XSRF-TOKEN cookie yet. Without it, the POST will return 419 (CSRF token mismatch). This follows the pattern documented in 55-RESEARCH.md Pitfall 2.
