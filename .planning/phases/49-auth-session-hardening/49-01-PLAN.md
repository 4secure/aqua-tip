    4. **ForgotPasswordController.php** (per D-01, D-02, AUTH-04): Replace the entire `__invoke` method body. The new implementation:
       - Looks up the user by email
       - Only calls `Password::sendResetLink()` if the user exists AND has no `oauth_provider`
       - Always returns HTTP 200 with the exact message: `"If an account exists with that email, a password reset link has been sent."`
       - No branching on status codes, no 422 responses, no provider name disclosure

       Full replacement for the method body:
       ```php
       public function __invoke(ForgotPasswordRequest $request): JsonResponse
       {
           $user = User::where('email', $request->email)->first();

           // Only send reset link to password-based users
           if ($user && !$user->oauth_provider) {
               Password::sendResetLink($request->only('email'));
           }

           // Always return identical response (anti-enumeration)
           return response()->json([
               'message' => 'If an account exists with that email, a password reset link has been sent.',
           ]);
       }
       ```

    5. **ResetPasswordController.php** (per D-03, AUTH-03): Add token and session nuclear wipe inside the `Password::reset` callback, AFTER `->save()` and BEFORE `event(new PasswordReset($user))`. Add `use Illuminate\Support\Facades\DB;` and `use Illuminate\Support\Facades\Log;` imports at the top of the file.

       The two lines to add inside the callback:
       ```php
       // Invalidate all tokens and sessions (per D-03)
       $user->tokens()->delete();
       DB::table('sessions')->where('user_id', $user->id)->delete();

       Log::info('Password reset: invalidated all tokens and sessions', ['user_id' => $user->id]);
       ```

       The full callback should become:
       ```php
       function ($user) use ($request) {
           $user->forceFill([
               'password' => Hash::make($request->password),
               'remember_token' => Str::random(60),
           ])->save();

           // Invalidate all tokens and sessions
           $user->tokens()->delete();
           DB::table('sessions')->where('user_id', $user->id)->delete();

           Log::info('Password reset: invalidated all tokens and sessions', ['user_id' => $user->id]);

           event(new PasswordReset($user));
       }
       ```
